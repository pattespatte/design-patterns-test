# E-postadress

## Använd mönstret när

Fråga bara efter mejladress när det behövs.

## Använd inte mönstret när

Beskrivning.

## Så fungerar mönstret

När du frågar användare om deras e-postadress måste du:

- göra det tydligt varför du frågar
- se till att fältet fungerar för alla dina användare
- hjälpa användare att ange en giltig e-postadress

Du kan också behöva kontrollera att användarna har tillgång till det e-postkonto de ger dig.

Gör klart vad e-postadressen ska användas till så att:

- användare känner sig säkra på att du inte kommer att missbruka det
- användare med flera e-postadresser kan välja vilken de ska ge dig

Om e-postadressfältet är en del av en inloggningsruta behöver du inte säga "Vi behöver din e-post så att vi kan logga in dig".

- Se till att fältet fungerar för alla dina användare
- Se till att fältet kan rymma upp till 254 tecken, vilket är det längsta en e-postadress kan vara.
